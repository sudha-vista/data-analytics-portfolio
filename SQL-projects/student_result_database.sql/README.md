# student_result_database

## Objective
Analyze student academic performance using SQL by evaluating subject-wise marks,
average scores, top performers, and department-level insights.

## Dataset
Two relational tables:
- **students** (student_id, name, department, semester)
- **results** (result_id, student_id, subject, marks)

## Tool Used
- MySQL Compiler

## SQL Concepts Used
- CREATE TABLE, INSERT
- PRIMARY & FOREIGN KEY
- JOIN
- GROUP BY
- Aggregate functions (AVG)
- WHERE, ORDER BY, LIMIT

## Project Files
- `analysis.sql` – SQL queries
- `output.txt` – Query results

## Key Insights
- Top scorer: **Sneha (Python – 95)**
- CSE department average: **85.25**
- ECE department average: **80.50**
- Anita and Sneha show consistent performance
